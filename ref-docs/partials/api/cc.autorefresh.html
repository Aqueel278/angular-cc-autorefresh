Collection of directives and controllers for scheduling a routine to run at intervals

ccAutoRefreshFn

  • The directive for those scenarios where you do not need a GUI but still require regular refresh of a value

ccAutoRefreshFnCtl

  • Controller to create a custom directive that extends ccAutoRefreshFn (eg with a GUI)
  • See ccAutoRefreshBtn for an example of a custom extension directive that uses this controller

ccAutoRefreshBtn

  • The directive provides a button for the user to pause, trigger a refresh and cancel a refresh that might be taking to long
