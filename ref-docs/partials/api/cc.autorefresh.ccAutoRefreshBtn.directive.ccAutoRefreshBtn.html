ccAutoRefreshBtn
directive in module cc.autorefresh.ccAutoRefreshBtn

Description

Schedule a expression to run every n number of milliseconds and assign the return value to the scope. Extends ccAutoRefreshFn with a button for the user to:

  • pause and refresh on demand
  • cancel a refresh in progress
Parameters

For details on all the parameters that this directive accepts see ccAutoRefreshFn

Usage

as element:
<cc-auto-refresh-btn
       cc-auto-refresh-fn="{expression}"
       refresh-interval="{expression}"
       refresh-model="{expression}">
</cc-auto-refresh-btn>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ccAutoRefreshFnexpression

An expression to schedule for execution.

refreshIntervalexpression

Angular expression that determines the interval (milliseconds) that refreshFn will be executed.

refreshModelexpression

Assignable angular expression to data-bind the value returned by refreshFn.
